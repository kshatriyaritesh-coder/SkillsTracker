# 📚 Skills Tracker

**Skills Tracker** is a student-focused productivity application designed to help learners build consistent study habits, track skill development, and stay motivated through progress visualization.

Whether you're learning programming, preparing for exams, or developing a new skill, Skills Tracker helps you stay focused and maintain consistency every day.

---

## ✨ Features

### 📊 Dashboard

* View today's study progress
* Track total study time
* Monitor active skills
* Check daily goals
* View current study streak

### 🎯 Skill Tracking

* Create and manage multiple skills
* Track progress for each skill
* Organize skills into categories
* Update learning milestones
* Monitor completion percentage

### ⏱️ Pomodoro Timer

* Built-in **25/5 Pomodoro Timer**
* Start, pause, resume, and reset sessions
* Automatic break timer
* Track completed focus sessions
* Record daily study time

### 🔥 Study Streaks

* Daily study streak
* Longest streak
* Weekly consistency tracking
* Motivation through continuous learning

### 📈 Progress Analytics

* Daily study statistics
* Weekly and monthly progress
* Study hour visualization
* Skill completion tracking
* Learning history

### 🎯 Daily Goals

* Set daily study targets
* Complete Pomodoro sessions
* Track personal learning objectives
* Monitor goal completion

### 👤 User Profile

* Personal information
* Total study hours
* Skills overview
* Current streak
* Longest streak
* Achievement summary

---

## 🚀 Why Skills Tracker?

Many students struggle with consistency rather than ability.

Skills Tracker focuses on helping students:

* Build daily study habits
* Stay accountable
* Track learning progress
* Visualize improvement over time
* Maintain motivation through streaks and statistics
